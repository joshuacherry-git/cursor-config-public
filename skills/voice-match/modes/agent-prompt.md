# Mode overlay: agent prompt

Loaded with `voice_profile.md` when `medium: agent-prompt`.

## Medium description

Cursor/coding-agent turns: **`/slash-skill` routers**, pasted heterogeneous context (URLs, traces, IGNORE blocks), **negative constraints**. Corpus **mtime-collapsed** + **April–May 2026** window + **high iterator share**.

## Deltas from core voice

- **Orchestration > chat:** Slash dispatch, pasted ground truth (`json`, traces), **REFERENCE partitions**.
- **`please` scarcity** (**core LLM-ingratiation low**)—issue commands cleanly.
- **Iterator micro-turns**: **`great`** / **`lets build`** compression; tighter **median sentence ~13–16w**, but **heavy right tail from initiator pastes**.
- **Negatives clustering:** **`don't`/`only`/cannot** coexistence clauses—guard production surfaces aggressively.
- **Epistemic stack** on causal/model critique (`cursor` **hedges ~11.7/1k**)— iterators drop toward **~4.8/1k** subset (`stats/cursor-agents.json` `by_role.iterator`).
- **Ellipsis caveat:** aggregated cursor corpus shows **high ellipsis/1k** driven by **May slice spikes + embedded excerpts** (**Pass B drift**)—do **not** treat ellipsis-heavy outputs as imitation target **`unless` rewriting quoted source containing them**.

## Opener patterns (anonymized)

1. **`/critical-review`** / **`/spec-refiner`** / **`/greenhouse-prep [role]`** leading.

2. **`run the critic again`** escalation.

3. **`What changed … ?`** paired infra questions.

4. **`build … summarize … compare`** stacked directives.

## Forbidden in this mode

- **Deferential softeners** duplicating SaaS etiquette (“Absolutely, I’d love to help…”).
- **Flattening pasted constraints**—preserve fences/verbatim config when user supplied as ground truth.
- Treating **`May ellipsis spike`** as evergreen author voice (**temporal_weakness**).
