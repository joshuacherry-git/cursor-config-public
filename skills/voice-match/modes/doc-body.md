# Mode overlay: doc body

Loaded with `voice_profile.md` when `medium: doc-body`.

## Medium description

Long-form Google-docs register: charters, specs, econ/mechanistic essays for **cross-functional** readers—**institutional `we`**, argumentative closure, slogan-grade anti-patterns when ethos matters.

## Deltas from core voice

- **Narrator default `we`** (core keeps **thinking moves**; pronoun dominance lands here—not in agent prompts/DM lowercase).
- **Pivot lexicon climbs:** **`Conversely`**, **`Put differently`**, **`on the other hand`**, **`This option/In this option`** option-scaffold.
- **Section hooks:** **`Why … ?`** then mechanism—not chatty reassurance.

**Pivot qualifier:** **`Conversely` / `Put differently` / `Why … ?`** hooks are **signature for charter-, design-argument-, and persuasive** doc passes. For **neutral PRD prose**, **status/reporting paragraphs**, short **definition stubs**, or **straight factual/industry-standard** exposition, prefer a **plain declarative spine** without invented pivots. **Heuristic:** if the input does **not** contain an argument or explicit tradeoff to flip, **do not manufacture one**.
- **Anti-pattern naming**: explicit memorable lines (“don't ship internal org-chart seams”).
- **`TL;DR` rare**—when present, compact exec stack with governance caveat (**same charter repeated excerpts exist—avoid duplicating mechanically**).

## Opener patterns (anonymized)

1. **`The purpose of this document …`**
2. **`Why is [system objective] behaving … ?`** → causal chain.
3. **`Context:` / `Challenge:` / `TL;DR` front-matter scaffolding.**
4. **`Takeover products` / definitional north-star opener.**
5. **Problem-gap:** *“Several factors contribute … limitations …”*

## Forbidden in this mode

- **`I think` scaffolding** dominating argumentative blocks (**personal `I`** clusters belong to **retro/promotion** excerpts—genre-specific).

## Exemplars

1. **`Why … ?` → belief → mechanism quoting operational metaphor (‘container’) → implication** (**hook discipline**).

2. **`Strongly recommend against …` decisive counsel after bounded caveat** (**stakes routing**).

3. **`Summarizing,` explicit takeaway tying fees to wedges/incidence economics** (**close move**).

4. **`Optimizer story first … wedge/deal caveat … Put differently—not fee recovery`** (**mechanism → reframing chain**).

5. **`Strongly recommend against …` contingent on insufficient evidence`** (**paired gate sentences**).
