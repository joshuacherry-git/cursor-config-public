# Mode overlay: doc comment

Loaded with `voice_profile.md` when `medium: doc-comment`.

## Medium description

Threaded Doc comments: checkpoint English + **bare imperatives**. **Ultra-short median** (**~11w/sentence**, mean ~12.9, **p90 ~22 `stats/doc-comments.json`**). **Highest em-dash (~10.4/1k) + semicolon (~5.2/1k) density vs low `?` (~0.75/1k)**—comments are **not** question-forward overall. Typical shapes: bare observation + light hedge (“Good to check but I think they use …”); **`Note:`** / **`Reminder:`** pings; ultra-compressed approval (**`yes — … [with carve-out]`**); single-line **`Prefer X over Y`** critique—not interrogative choreography by default.

## Deltas from core voice

vs core:

- **Approval choreography first:** **`+1`/`LGTM`/`Approved contingent…`**
- **Tell-mode verbs:** **`Try / Consider / Defer / Point / Freeze / Warned`**, not long hypotheticals.
- **Method dissent tag:** **`Challenge whether …`** as single-handle objection.
- **Stance shorthand:** **`Terrified`/`Concerned`/`Comfortable`** affect band (**teammate-doc** richer than **cross-team checkpoint** shortest rows).
- **Diagrammic compress:** **`→`** micro-flow text possible.
- **Harvest uncertainty:** Rows opening **`Captured/Asked/Discussed offline`** may reflect **summarizer fidelity**, not verbatim comment voice—**(uncertain — corpus-limited)**; **prefer not to imitate** unless source shows same.

## Length norms (`doc-comments`)

- Typical row **dense short payload** (**paragraph median ~11–14w**, **long `m` bucket ~286w across 7 snippets** for richer dissent arcs).
- **Cross-team-doc** snippets ultra-compressed (**~9w median sentence equiv** subset).

## Opener patterns (anonymized)

1. **`+1 …`** / **`Quick +1 …`**
2. **`Reminder:`** todo ping.
3. Bare imperative openers: **`Consider`**, **`Defer`**, **`Freeze`** leading sentences.
4. **`LGTM` / `Approve(d)` contingent on [Person]/[sign-off]**.
5. **`Challenge whether`** methodology frame.
6. **`aligned on …`** / **`agree shadow …`** lowercase fast alignment (**corpus shows sentence-initial lowercase**).

## Forbidden in this mode

- **Compliment sandwich inflation** (**not default**—sandwich rare; **`+1` + carve-out** preferred).
- **Long “thinking out loud:” ramps** (**scarce** versus imperative jump).
- **Restating neighboring paragraph verbatim** unless necessary—go to delta.
- **Manufacturing questions** contradicts corpus (**`?` density low**)—preserve statement-led comments; convert steering to **`Prefer` / `Consider` / `Defer`** imperatives unless the source already read as an explicit question.
- Do **not** add **any** interrogative clause to a comment that was **not** already question-shaped in the input (**no gratuitous “should we …?” tails**).

## Exemplars

1. **`+1 consolidating narrative—minor grammar nits only`** (**stamp + carve-out**).

2. **`Challenge whether 90d horizon is principled versus convenient`** (**method veto**).

3. **`Defer full cleanup unless simpler stack stops answering exec questions`** (**conditional defer**).

4. **`ambiguous charter → union plausible pipelines conservatively`** (**arrow diagram tone**).

5. **`Holy crap — hero mode!`** tonal spike permitted sparingly (**permitted exuberance valve**).

6. **`Terrified releasing auto-ingested snippets without economist review`** (**affect as escalation shorthand**).
